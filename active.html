php使用array_keys,array_values来构造sql插入语句

  javascript   刘孟磊   浏览(7)    评论(7)   2天前 

在开发接口的过程中，会经常的使用curl发送post请求，以及将返回的json字符串转化为php数据的方法。因为自己现在主要就是负责公司的app接口工作，所以对于接口中常使用到的方法产生了自己整理下的想法，避免总是重复的去写。下面，是我整理好的两个函数，分别是curl发送post请求和json字符串转化php数据。喜欢的朋友可以直接的拷贝，省的自己再去写。 
/*//当我们的表里面有这些字段的时候
username = $username  // 用户名
mobile = $mobile  //手机号
sex = $sex  // 性别
addr = $addr//地址
pwd = $pwd //密码
...等等
*/

//如果是正常的写法，那肯定是这样的
//1、反正我个人是不喜欢这种写法的
$sql = "insert into table ('$username','$mobile','$sex','$addr','$pwd') value (username,mobile,sex,addr,pwd)";

//2、看下这么写
$data = array(
    'usernaem' => $username,
    'mobile'   => $mobile,
    'sex'    => $sex,
    'addr'  => $addr,
    'pwd' => $pwd,
);

$keys = '`' . join('`,`',array_keys($data)) .'`';
$values = '"' . join('","',array_values($data)) . '"';
$sql = "insert into table ($keys) values ($values)";
